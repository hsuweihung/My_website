Unity 2D  問答遊戲

Unity 2D 問答遊戲
柯基犬大冒險

專案摘要

使用Unity 遊戲引擎進行遊戲開發，利用問答遊戲讓玩家能多深入了解與柯基犬有關的知識。

專案負責項目：​遊戲發想、腳本撰寫、C#程式碼撰寫、遊戲場景製作(素材組合)、RPG 人物製作 

遊戲機制

1. RPG 問答遊戲
2. 系統給予玩家破關需要的資訊
3. 累積分數已兌換獎勵
4. 答錯需重新作答
5. 分數累積到2000分即可獲勝

遊戲 Demo 影片

心得

第一次接觸 Unity 就要開發出一個小遊戲，期間遇到了蠻多關卡，除了程式碼deBug的循環外，也要一直去思考該怎麼寫才能寫出我想要的遊戲互動形式。 這次專案也讓我對Unity 有所認識外，也算是第一次去針對一項功能思考如何撰寫程式碼。
