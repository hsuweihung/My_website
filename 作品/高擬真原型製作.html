高擬真原型製作

Framer X 原型製作
第一銀行 & Lawsnote

專案摘要

利用 Framer X 進行高擬真 Prototype的設計，遇到的挑戰是要透過 Code Component 製作出不同的功能，對React 框架並不熟悉，因此上網找了jQuery 的動畫效果做參考，並參照 Framer X 本身的 API 文件，最後和 PM 一起把原型完成。也因為這次的專案，讓我有了第一次看 API 文件的經驗。透過這項專案也讓我對 Java Script 有了認識，並嘗試用不同工具產出原型，也讓我更有耐心去了解程式碼的邏輯。

第一銀行

 專案負責項目：透過PM 的簡介，了解此款App的功能及客戶需求，依照需求利用Code Component 做出點擊Tab bar 切換分頁及WebView的功能。

Tab bar 切換分頁功能

透過程式碼的方式，切換Tab bar 效果不再需要畫出所有分頁，而是利用平移位置的改變，達到切換分頁的效果

WebView 功能

針對頁面內容除了利用Design Component 達到scorll 效果外，點擊內容可跳到詳細文章頁一樣利用了平移位置的改變，達到此效果

Lawsnote 七法

 專案負責項目：​透過PM 的簡介，了解每層layer 需完成的功能，依照功能完成切換分頁及第二層layer切換( 點選呈現不同內容且會隨者內容上下移動)之功能。

切換分頁呈現不同內容

針對未處理、已通知等分類，讓使用者可隨意選取想顯示的內容至相關內規區域中顯示，且當點選的分類時，提醒會消失。

點選呈現不同內容且會隨者內容上下移動

使用者依據法令屬性的，分別顯示出變更法規內容，且Layer會跟著內容上下移動，右邊的數字（代表變更法條數量）會消失。
